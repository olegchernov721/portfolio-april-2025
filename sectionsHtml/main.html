О черновике вкратце

В нём я реализовал одностраничное приложение (SPA), где контент загружается динамически без перезагрузки страницы. Это позволяет сайту работать быстрее и создавать более плавный пользовательский опыт.

Основной акцент был сделан на удобную навигацию и оптимизацию загрузки контента. Когда пользователь кликает на пункт меню, с помощью JavaScript выполняется запрос fetch() к нужному HTML-файлу (или API), и содержимое раздела подставляется в главный контейнер страницы. Благодаря этому реализована быстрая смена разделов, без перехода на другую страницу.

Структура проекта разбита на модули, для каждого раздела можно подключить отдельный HTML-файл, который подгружается при необходимости. Это облегчает поддержку проекта и позволяет гибко изменять контент без переписывания всей страницы.
